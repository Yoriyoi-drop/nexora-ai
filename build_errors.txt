# Bagian penting yang aku tambahkan

## 1. SemanticDedupFilter

Ini upgrade besar dari dedup biasa.

Karena internet modern penuh:

* repost
* paraphrase
* AI-generated duplication
* synthetic spam

### Exact hash sudah tidak cukup.

Tambahkan:

```rust
SemanticDedupFilter::new(
    EmbeddingModel::MiniLM,
    similarity_threshold = 0.92,
)
```

Flow:

```text
text
 ↓
embedding
 ↓
vector similarity
 ↓
duplicate detection
```

Ini jauh lebih kuat.

---

# 2. ToxicityFilter

Karena kalau kamu streaming langsung dari internet:

* bakal ada racism
* malware snippets
* jailbreak prompt
* spam SEO
* forum insanity

Manusia meninggalkan jejak digital seperti rakun hiperaktif diberi akses keyboard.

```rust
.add(ToxicityFilter::new(
    threshold = 0.80,
    model = ToxicityModel::FastBert
))
```

---

# 3. PromptInjectionFilter

Ini penting untuk future agentic Nexora.

Filter:

* "ignore previous instruction"
* "you are chatgpt"
* hidden system prompt
* jailbreak attempts

Karena nanti dataset internet bakal penuh data poisoning terhadap AI.

---

# 4. EntropyFilter

Perplexity bagus.
Entropy juga penting.

Detect:

* repetitive text
* token loops
* garbage generation
* synthetic low-information content

```rust
.add(EntropyFilter::new(
    min_entropy = 3.5
))
```

---

# 5. DomainClassifier

Ini bagian yang bikin Nexora beda.

Sample otomatis diarahkan ke subsystem.

```text
code          → CODE-SENTINEL
reasoning     → ORACLE-7

architecture  → ARCH-WEAVER
planning      → META-REASONER
```

Implementasi:

```rust
enum Domain {
    Code,
    Reasoning,
    Memory,
    Architecture,
    Science,
    General,
}
```

---

# 6. CurriculumRanker

Sangat penting.

Model jangan langsung makan data sulit.

Urutan:

1. grammar sederhana
2. instruction basic
3. reasoning medium
4. chain-of-thought
5. agentic planning
6. multi-hop logic

AI training itu seperti pendidikan manusia. Bedanya AI tidak bolos sekolah demi main Mobile Legends.

---

# 7. TrustScoreFilter

Dataset source diberi reputasi.

```text
Wikipedia      = 0.95
Arxiv          = 0.97
Random forum   = 0.41
SEO farm       = 0.12
```

Jadi Nexora belajar prioritas sumber terpercaya.
