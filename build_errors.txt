File Kunci yang Harus Dibuat Pertama
Urutan pembuatan yang benar:
1. crates/foundation/src/shared/base_model.rs
   → Trait NxrModel: identity(), capabilities(), agents(), architecture()

2. crates/foundation/src/shared/model_identity.rs
   → enum NxrModelId, struct ModelMeta, enum Tier

3. crates/foundation/src/shared/capability_spec.rs
   → struct CapabilitySpec, domain declarations

4. crates/foundation/src/models/mod.rs
   → pub mod untuk semua 10 model

5. crates/foundation/src/models/omnis/identity.rs
   → Implementasi pertama, jadi template untuk 9 lainnya

Kenapa Tidak Satu Folder Per Model di Root crates/
Karena model bukan aplikasi mandiri. Model adalah konfigurasi + arsitektur + agent spec yang berjalan di atas infrastruktur yang sama. Yang berbeda antar model hanyalah:
├── architecture.rs   → cara neuron tersusun
├── agents.rs         → agen mana yang aktif
├── config.rs         → hyperparameter
└── capabilities.rs   → apa yang bisa dilakukan
Sisanya — inference engine, memory, tokenizer, runtime — shared oleh semua model. Memisahkan ke root crates/ berarti duplikasi kode yang tidak perlu dan maintenance nightmare.
